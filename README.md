# HouseRentalSystem (房屋租赁管理系统)


-  基于layui+SSM写的毕业设计,系统界面仿Q房网UI进行排版。后台管理界面使用的layui自带后台管理样式。后端语言使用的目前主流的Spring+SpringMvc+Mybatis框架。实现了一系列基本的增删改查和权限控制功能。

功能设计：
1. 房屋信息展示
2. 房屋详情页面展示
3. 用户登录注册
4. 用户发布房源信息
5. 用户管理自己发布的所有房源信息
6. 修改密码
7. 管理员登录
8. 管理员发布房源信息
9. 管理员管理所有用户信息
